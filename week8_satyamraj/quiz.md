# Agentic AI - Short Quiz

## Questions

1. What is an agentic AI system?
   a) A system that follows pre-programmed instructions only
   b) A system that can reason, plan, and act autonomously to achieve goals
   c) A system that only processes data in batches
   d) A system that requires constant human supervision for every task

2. Which of the following is a key characteristic of agentic AI?
   a) Lack of goal-oriented behavior
   b) Ability to adapt to changing environments
   c) No memory of past actions
   d) Inability to use tools

3. What is a "tool" in the context of agentic AI?
   a) A physical robot arm
   b) A function or API that the agent can use to interact with the world
   c) A programming language
   d) A database management system

4. Which framework is commonly used for building agentic AI systems?
   a) TensorFlow
   b) PyTorch
   c) LangChain
   d) Flask

5. What is the role of a "planner" in an agentic pipeline?
   a) Executes tasks directly
   b) Creates a step-by-step plan to achieve a goal
   c) Stores data permanently
   d) Only answers questions

---

## Answers
1. b
2. b
3. b
4. c
5. b
